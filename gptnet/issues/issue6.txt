# Issue #6: Full Model Orchestration & Output Head
**Assignee:** Lead Architect

## Description
Stack 12 blocks and add the final normalization and un-embedding (Linear Head).

## Acceptance Criteria
​[ ] Model contains exactly 12 transformer blocks.
​[ ] Final output tensor shape is [Batch, SeqLen, 50257].
​[ ] Total model parameters are approximately 124 million.

## Required Class Structure
```csharp
public class GPT2Model : nn.Module<Tensor, Tensor> {
    private readonly GPT2Embeddings embeddings;
    private readonly nn.ModuleList<TransformerBlock> blocks;
    private readonly nn.Module<Tensor, Tensor> ln_f; // Final LayerNorm
    private readonly nn.Module<Tensor, Tensor> lm_head; // Linear Vocab Head

    public override Tensor forward(Tensor x) {
        x = embeddings.forward(x);
        foreach (var block in blocks) x = block.forward(x);
        x = ln_f.forward(x);
        return lm_head.forward(x); // Returns Logits
    }
}
